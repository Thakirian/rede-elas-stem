# ⚙️ Requisitos Não-Funcionais (RNF)

Diretrizes de qualidade e restrições técnicas para o desenvolvimento.

| ID | Descrição | Categoria |
| :--- | :--- | :--- |
| **RNF01** | A interface deve ser projetada para dispositivos móveis (Android/iOS). | Portabilidade |
| **RNF02** | O sistema deve possuir interface intuitiva seguindo as 10 Heurísticas de Nielsen. | Usabilidade |
| **RNF03** | O sistema deve garantir a privacidade total, desvinculando dados em posts anônimos. | Segurança |
| **RNF04** | O design deve possuir contraste e tipografia acessíveis seguindo padrões WCAG. | Acessibilidade |
| **RNF05** | O tempo de carregamento entre transições de tela no protótipo deve ser fluido. | Performance |
