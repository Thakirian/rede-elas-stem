# Histórias de Usuário (User Stories)

Detalhamento das necessidades das usuárias e critérios de aceite para validação.

## 1. Infraestrutura e Acesso (Core)

| ID | Persona | História de Usuário (User Story) | Critério de Aceite |
| :--- | :--- | :--- | :--- |
| **US01** | Visitante | Como estudante da UTFPR, desejo me cadastrar no app usando meu e-mail institucional... | Validação de domínio @alunos.utfpr.edu.br. |
| **US02** | Usuária | Como usuária cadastrada, desejo realizar login com senha... | Tela de login com fluxo de recuperação de senha. |

## 2. Módulo: Elas Inspiram

| ID | Persona | História de Usuário | Critério de Aceite |
| :--- | :--- | :--- | :--- |
| **US05** | Estudante | Como usuária, desejo navegar por uma galeria de perfis inspiradores... | Lista com cards contendo foto, nome e cargo. |
| **US06** | Estudante | Como usuária, desejo salvar perfis inspiradores como "favoritos"... | Ícone de "favoritar" que salva o perfil. |

## 3. Módulo: Conexões (Mentoria)

| ID | Persona | História de Usuário | Critério de Aceite |
| :--- | :--- | :--- | :--- |
| **US07** | Caloura | Como aluna, desejo filtrar mentoras por disciplinas específicas... | Campo de busca e filtro por disciplina. |
| **US08** | Caloura | Como aluna, desejo enviar uma solicitação de mentoria para uma veterana... | Botão "Solicitar Ajuda" com notificação. |
| **US09** | Veterana | Como mentora, desejo aceitar ou recusar solicitações de mentoria... | Painel de gestão no perfil da mentora. |

## 4. Módulo: Comunidade

| ID | Persona | História de Usuário | Critério de Aceite |
| :--- | :--- | :--- | :--- |
| **US10** | Usuária | Como usuária, desejo visualizar uma agenda centralizada de eventos... | Lista cronológica com data e local. |
| **US11** | Usuária | Como usuária, desejo salvar um evento no meu calendário pessoal do app... | Botão "Salvar na Agenda". |

## 5. Módulo: Acolhimento

| ID | Persona | História de Usuário | Critério de Aceite |
| :--- | :--- | :--- | :--- |
| **US12** | Usuária | Como usuária, desejo postar relatos ou dúvidas no fórum de forma anônima... | Checkbox "Postar Anonimamente" no formulário. |
| **US13** | Admin | Como administradora, desejo moderar ou remover postagens inadequadas... | Acesso administrativo para exclusão. |
